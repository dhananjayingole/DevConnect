DevConnect is a powerful backend API platform that connects developers with recruiters, enabling seamless job searching, profile management, and recruitment processes. Built with FastAPI, it provides robust authentication, job posting, and developer discovery features.

## ✨ Features

### 👨‍💻 For Developers
- **User Registration & Authentication** - Secure JWT-based authentication
- **Developer Profile Management** - Create and update professional profiles
- **Skill Management** - Add skills to your profile for better discoverability
- **Job Search** - Browse and filter jobs by type, location, and skills
- **Save Jobs** - Bookmark interesting positions for later
- **Profile Analytics** - Track profile views and job applications

### 🏢 For Recruiters
- **Company Profile** - Create and manage company information
- **Job Posting** - Create, update, and close job listings
- **Candidate Discovery** - Search and filter developers by skills
- **Application Management** - Track job applications (coming soon)

### 👑 For Admins
- **Platform Analytics** - View platform statistics and metrics
- **User Management** - Manage user accounts and permissions
- **Content Moderation** - Monitor and moderate platform content
